# Brijal Savaniya — Influencer Website

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000

## Routes

- `/` — high-converting landing page
- `/media-kit` — shareable media kit page
- `/media-kit/print` — print-friendly view (use browser Print → Save as PDF)

## Update influencer details

Edit:

- `src/data/site.ts`

Set real:

- Instagram URL
- WhatsApp number
- Email
- Metrics and audience breakdown
- Brand list and content showcase items
